
### 待辦清單Todolist

* 網站範例(Demo):http://todolist-demo.surge.sh
* 可以新增「優先事項」、「雜事」任務
* 可檢視完成的任務
* 可以刪除個別與所有任務
* 可以計算剩餘任務筆數


#### 嘗試使用的技術

* 基礎技術：HTML+CSS
* 進階技術：VueJS、Firebase
* 其他：Markdown語法＋Giphycapture(製作GIF圖檔)+Npm Surge(用來上傳Demo網站）＋Git
